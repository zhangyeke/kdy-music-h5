# kdy-music-h5

### **项目安装依赖**

```
//推荐node版本18+
npm install
```

### 运行开发环境

```
npm run dev
```

### 打包发布

```
npm run build
```

### ✨TIP

在项目根目录新建.env.development和.env.production文件

```.env.development
//.env.development
VITE_API_URL = /api // 你的开发环境请求地址
VITE_APP_TITLE = //项目名称
VITE_BASE_PATH = / //放置开发环境项目的根目录
```

```
//.env.production
VITE_API_URL = /api // 你的生成环境请求地址
VITE_APP_TITLE = //项目名称
VITE_BASE_PATH = / //放置生成环境项目的根目录'
```
